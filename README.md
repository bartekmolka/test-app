Simple colors app
---

## Features:
- adding colors ➕
- removing colors ➖
- filtering colors by: 🗃️
	- red 🔴
	- green 🟢
	- blue 🔵
	- saturation 🌈
- saving colors in localStorage 📁

## Running app 💻

To run app type:

```
npm start
```

You can now open app [here](http:localhost:3000) 🥳
